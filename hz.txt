			标签管理 :
查看所有标签  git tag
创建标签      git tag name
指定提交信息  git tag -a name -m “comment”
删除标签      git tag -d name

标签发布      git push origin name
Echo “new code ” >> demo.txt   信息添加
Git status    查看状态
Git add demo.txt   提交暂存区
Git commit -m ‘new code commit’  备注
Git push   推送
Git tag    查看标签
Git tag haoyun  创建标签
Git push origin haoyun  推送同步到github上边

			分支
查看分支：git branch
创建分支：git branch 名字
切换分支：git checkout 分支名称
合并代码：git merge 分支名称
删除分支: git branch -d 分支名称
